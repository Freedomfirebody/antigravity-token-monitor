# Token Monitoring

**상태**: verified

## 개요

Antigravity 코딩 세션의 토큰 사용량을 실시간으로 수집, 분석, 시각화합니다.

## 핵심 기능

### 1. 실시간 토큰 추적
- Antigravity `language_server` 프로세스를 자동 탐지
- 내부 RPC로 세션별 토큰 메타데이터 수집
- 폴링 주기: 기본 60초 (설정 가능)

### 2. 히스토리 세션 복구
- 익스텐션 실행 이전 생성된 세션도 RPC fallback으로 복구
- 파일시스템 세션 ID 기반 직접 조회

### 3. 대시보드
- KPI 카드: 총 토큰, 세션 수, 비용
- 180일 활동 히트맵
- 모델별 사용량 테이블
- 세션 리더보드 + 미니 차트
- 실시간 리프레시 카운트다운

### 4. 비용 추정
- LiteLLM 오픈소스 가격표 기반
- 모델별 카테고리별 비용 산정 (input, output, cache, reasoning)

### 5. 상태바
- VS Code 하단에 총 토큰 수 상시 표시
- 클릭 시 대시보드 오픈

## 수용 기준

- [ ] Antigravity 프로세스 탐지 후 토큰 데이터가 대시보드에 표시됨
- [ ] RPC 불가 시 estimated 모드로 fallback
- [ ] 다중 VS Code 인스턴스에서 중복 처리 없음
- [ ] 비용 추정치에 "참고용" 경고 표시
