# Product Sense

## 대상 사용자

Antigravity를 사용하는 개발자로, 자신의 AI 코딩 세션에서 **토큰 사용량과 비용을 파악**하고 싶은 사람.

## 사용자 멘탈 모델

1. "오늘 Antigravity로 얼마나 많은 토큰을 썼지?" → **상태바** (즉각 확인)
2. "세션별로 어떤 모델을 가장 많이 사용했지?" → **대시보드 모델 테이블**
3. "이번 달 비용이 얼마나 되지?" → **KPI 카드 + 히트맵**
4. "가장 토큰을 많이 쓴 세션은?" → **세션 리더보드**

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **투명성** | 토큰이 어디에, 얼마나 사용되는지 명확히 보여줌 |
| **비침투성** | 백그라운드에서 자동 수집, 작업 흐름을 방해하지 않음 |
| **정직함** | 추정 데이터는 `estimated`로, 정확한 데이터는 `reported`로 명시 |
| **안전성** | 로컬 전용, 민감 데이터 기본 비수집 |

## 우선순위 프레임워크

1. **정확성** > 기능 수 — 잘못된 토큰 수치보다 정확한 수치가 중요
2. **안정성** > 새 기능 — RPC 변경에 깨지지 않는 graceful degradation
3. **로컬 우선** > 클라우드 연동 — 외부 의존 최소화
4. **비공식임을 인지** — Google의 공식 도구가 아님을 항상 명시
